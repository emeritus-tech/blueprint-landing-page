Checkbox

Create consistent cross-browser and cross-device checkboxes with our completely rewritten checkbox component.


Checkbox without label

<input type="checkbox" class="form-check-input" aria-label="Default Checkbox">
<input type="checkbox" class="form-check-input" aria-label="Active Checkbox" checked>
<input type="checkbox" class="form-check-input" aria-label="Disabled Checkbox" disabled>
<input type="checkbox" class="form-check-input" aria-label="Disabled Active Checkbox" checked disabled>

Indeterminate Checkbox without label

Checkboxes can utilize the Indeterminate class when manually set via JavaScript (there is no available HTML attribute for specifying it).

<input type="checkbox" class="form-check-input-minus" aria-label="Minus Checkbox">
<input type="checkbox" class="form-check-input-minus" aria-label="Minus Active Checkbox" checked>
<input type="checkbox" class="form-check-input-minus" aria-label="Minus Disabled Checkbox" disabled>
<input type="checkbox" class="form-check-input-minus" aria-label="Minus Disabled Checkbox" checked disabled>
